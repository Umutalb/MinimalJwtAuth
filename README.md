# MinimalJwtAuth

A lightweight and reusable JWT authentication service built with .NET.

This project provides:
- User registration
- Secure password hashing (SHA256)
- JWT-based login system
- Token expiration and claims
- Minimal, clean architecture for reuse in any backend project

## Usage

Clone the repository and run:

- .NET version: .NET 10
- IDE: Visual Studio 2026

Clients can interact with the API through:
- POST /register
- POST /login

This repository is designed as a clean starting point for authentication in other .NET projects.
